How to import a workflow from the Workflow Hub and run it on remote COVID-19 data

»
Mar 25, 2021
training FAQ COVID-19

Thanks to a great Collaboration between the Galaxy Community, Dockstore and the Workflow Hub, it is also possible to import a workflow directly into Galaxy and run it. This is using the GH4GH API TRS.

We have created a video showcasing the process for a COVID-19 data analysis workflow:

  1. Search in the Workflow Hub and retrieve a Galaxy workflow.
  2. Import it easily into Galaxy.
  3. Get remote COVID-19 data into your Galaxy history.
  4. Run the imported workflow on the remote data.
