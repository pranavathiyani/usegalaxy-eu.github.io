The European Galaxy server listed as an official resource on the COVID-19 Data Portal

»
Feb 5, 2021
COVID-19


The COVID-19 Data Portal emerged as an effort to coordinate data sharing to speed up the coronavirus research. It brings together relevant datasets submitted to EMBL-EBI and other major centres for biomedical data and constitutes an updated resource to which scientist can upload access and analyse reference data.

The European Galaxy Server is now listed as part of the “Computing support resources”. Under the “Analysis tools”, the COVID-19 analysis on Galaxy Server can be found, and the COVID-19 analysis on Galaxy under “Standard for data sharing”. An “ELIXIR publications” is also listed: No more business as usual: Agile and effective responses to emerging pathogen threats require open data and open analytics.


A few updates about our #COVID19 work in our new blog post: https://t.co/NfB0Npe97u

New tools and new workflows included e.g. combine the ARTIC variant calling workflow with the consensus sequence building one and be ready to submit your data to public archives.
